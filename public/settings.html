Search Tags

By default, search terms entered into the search field only match title and tags. You can apply the following tags to further specify where to search:

  • all:searchterm Search in title, tags, and text.
  • title:searchterm Search only in title.
  • tags:searchterm Search only in tags.
  • file/files:searchterm Search for files.

Publish a Wiki

All wiki entries with tag publish are going to be converted to HTML and stored in /public/publish.

In order to publish the wiki online, create the following folder structure on your server:

  • wiki
    • css
      • bootstrap-grid.min.css
      • prism.css
      • styles.css
    • img
      • favicon.ico
    • js
      • functions.js
      • index.js
      • marked.min.js
      • prism.js
      • view.js
    • publish
      • all published files that are generated when clicking "Publish Wiki"
    • index.php

Where index.php contains the following code:

<?php
require('./publish/' . $_GET['slug'] . '.php');
?>
